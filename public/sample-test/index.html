Sample Norfolk Publication

This is a sample HTML document demonstrating the Norfolk publishing platform capabilities. It includes proper metadata that will be automatically extracted by the publishing system.

Features Demonstrated

  • Title extraction from the <title> tag
  • Description extraction from meta description
  • Keywords extraction from meta keywords
  • Automatic slug generation from the title

Publishing Process

When you publish this file using the Norfolk publishing skill, it will:

  1. Extract the title "Sample Norfolk Publication"
  2. Generate the slug "sample-norfolk-publication"
  3. Use the meta description as the page description
  4. Extract tags from the keywords meta tag
  5. Apply Norfolk AI Labs branding automatically
  6. Deploy to share.norfolkailabs.com

Usage Examples

You can publish this file using:

# Minimal (all metadata auto-detected)
python3 scripts/publish.py sample-content.html

# With Atlas
"Publish sample-content.html to Norfolk"

# With custom parameters
"Publish sample-content.html with title 'Custom Title' and tags 'demo,test'"

The published page will be available at https://share.norfolkailabs.com/sample-norfolk-publication/ and listed on the homepage index.
