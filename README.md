# Norfolk AI Labs - Share Pages

Branded publishing platform for Atlas-generated content.

## Overview

This repository powers `share.norfolkailabs.com` - a simple publishing platform for HTML content generated by Atlas. It's designed as a lightweight "artifact publisher" rather than a full CMS.

## Architecture

- **Hosting:** Cloudflare Pages
- **Source:** This GitHub repository
- **Workflow:** Atlas generates HTML → publish script → branded URL
- **Deployment:** Automatic via git push to main branch

## Publishing Content

### Quick Start

```bash
# Publish an HTML file
./scripts/publish.py tutorial.html --slug my-tutorial --title "My Tutorial"

# With description and tags
./scripts/publish.py guide.html \
  --slug setup-guide \
  --title "Setup Guide" \
  --description "Complete setup instructions" \
  --tags "guide,setup,tutorial"
```

### Publishing Workflow

1. **Generate content** - Atlas creates HTML file
2. **Run publish script** - Copies to repo with branding
3. **Automatic deployment** - Cloudflare Pages deploys from git push
4. **Live URL** - Content available at `share.norfolkailabs.com/slug/`

### Script Options

- `--slug` (required) - URL slug for the content
- `--title` - Content title (auto-detected from HTML if not provided)
- `--description` - Description for homepage listing
- `--tags` - Comma-separated tags
- `--no-push` - Create locally without git push (for testing)

## Development

### Local Testing

```bash

./scripts/publish.py test.html --slug test-content --no-push

python -m http.server 8000 --directory public
# Open http://localhost:8000
```

### Requirements

- Python 3.7+
- Git configured with GitHub access
- Write access to this repository

## Deployment

Deployment is automatic via Cloudflare Pages:
1. Push to `main` branch triggers build
2. Cloudflare deploys `public/` directory
3. Content available at `share.norfolkailabs.com`
4. Deployment typically takes 1-2 minutes

## Design Principles

1. **Artifact publishing, not content management** - Designed for finished HTML, not authoring
2. **Minimal maintenance** - Simple structure, proven infrastructure
3. **Branded consistency** - All content gets Norfolk AI identity
4. **Fast publishing** - From HTML file to live URL in under 60 seconds
